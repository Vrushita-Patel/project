# GitHub Clone

A simple frontend clone of GitHub built with HTML, CSS, and JavaScript.  
This project recreates the look and flow of GitHub’s landing page, sign-in, sign-up, and dashboard screens.

## ✨ Features

- Responsive landing page similar to GitHub
- Sign up and sign in pages
- Dashboard UI with sample statistics and repository cards
- Local storage-based demo authentication
- Clean and modern styling inspired by GitHub

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📁 Project Structure

- `index.html` — Home page
- `signin.html` — Sign in page
- `signup.html` — Sign up page
- `dashboard.html` — User dashboard
- `styles.css` — Main styles
- `github-styles.css` — Additional GitHub-themed styling
- `script.js` — JavaScript logic
